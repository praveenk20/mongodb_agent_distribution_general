MongoDB Agent AI - Installation

üì¶ Installation

Prerequisites

  • Python: 3.10 or higher
  • pip: Package installer for Python
  • MongoDB: Access to MongoDB instance (optional for initial setup)
  • LLM API Key: OpenAI, Anthropic, or Azure OpenAI
üí° Tip: Check your Python version: python3 --version

üöÄ Option 1: Quick Setup (Recommended)

Use the interactive setup script for guided installation:

# Navigate to distribution directory
cd mongodb_agent_distribution_general

# Run setup script
chmod +x setup.sh
./setup.sh

The setup script will:

  1. Check Python version (3.10+ required)
  2. Create and activate virtual environment
  3. Install the wheel package
  4. Guide you through configuration
  5. Verify the installation
‚úÖ Expected Output: 
‚úÖ MongoDB Agent installed successfully!
‚úÖ Configuration file created: .env
‚úÖ Server ready to start: python3 -m mongodb_agent.api

üìã Option 2: Manual Installation

Step 1: Create Virtual Environment

# Create virtual environment
python3 -m venv venv

# Activate virtual environment
# macOS/Linux:
source venv/bin/activate

# Windows:
venv\Scripts\activate

Step 2: Install Wheel Package

# Navigate to distribution directory
cd mongodb_agent_distribution_general

# Install the wheel
pip3 install mongodb_agent_ai-1.0.0-py3-none-any.whl
üì¶ Package Size: ~70KB wheel file contains all dependencies

Step 3: Verify Installation

# Check installed packages
pip3 list | grep mongodb-agent-ai

# Expected output:
# mongodb-agent-ai    1.0.0

üîÑ Reinstalling or Upgrading

If you need to reinstall or upgrade the package:

Using Reinstall Script:

cd mongodb_agent_distribution_general
chmod +x reinstall.sh
./reinstall.sh

Manual Reinstall:

# Uninstall old version
pip3 uninstall mongodb-agent-ai -y

# Install new version
pip3 install --force-reinstall mongodb_agent_ai-1.0.0-py3-none-any.whl

üîß Troubleshooting Installation

Issue: "No module named 'mongodb_agent'"

Solution:

  1. Ensure virtual environment is activated: source venv/bin/activate
  2. Reinstall: pip3 install --force-reinstall mongodb_agent_ai-1.0.0-py3-none-any.whl
  3. Check Python path: python3 -c "import sys; print(sys.path)"

Issue: "Permission denied"

Solution:

# Don't use sudo with pip in virtual environment
# Instead, ensure venv is activated:
source venv/bin/activate
pip3 install mongodb_agent_ai-1.0.0-py3-none-any.whl

Issue: Wheel file not found

Solution:

# Verify wheel file exists
ls -lh mongodb_agent_distribution_general/*.whl

# Expected: mongodb_agent_ai-1.0.0-py3-none-any.whl
